# PR for Official MCP Servers List

## Steps to Submit

1. Go to: https://github.com/modelcontextprotocol/servers
2. Click "Fork" to create your copy
3. Edit the README.md file
4. Find the appropriate section (likely "Community Servers" or create under a "UX/Design" category)
5. Add your entry (see below)
6. Submit a Pull Request

## PR Description

```markdown
## Summary

This PR adds the UX MCP Server to the community servers list.

## About UX MCP Server

A comprehensive MCP server providing UX best practices, accessibility guidelines, and design system knowledge to AI assistants.

### Features

- **28 Knowledge Resources**: WCAG accessibility, Nielsen heuristics, design systems, e-commerce patterns, voice UI, AR/VR, healthcare UX, finance UX, and more
- **23 Analysis Tools**: Accessibility audits, color contrast checking, typography scales, form pattern recommendations, dark pattern detection
- **4 Workflow Prompts**: Pre-configured review workflows for accessibility, usability, and design systems

### Compatibility

Works with all MCP clients:
- Claude Desktop
- Claude Code (CLI)
- Cursor IDE
- Continue.dev
- Cline (VS Code)
- Zed Editor

### Installation

```bash
npm install -g @elsahafy/ux-mcp-server
```

### Links

- **GitHub**: https://github.com/elsahafy/ux-mcp-server
- **npm**: https://www.npmjs.com/package/@elsahafy/ux-mcp-server
- **Documentation**: https://github.com/elsahafy/ux-mcp-server#readme

### Checklist

- [x] Server is open source
- [x] Published on npm
- [x] Comprehensive documentation
- [x] Works with multiple MCP clients
- [x] Actively maintained
```
